Colour Variable Reference

Edit the CSS variables at the top of this file to test different colour combinations.

  • --bg-primary: Main background colour
  • --bg-secondary: Secondary background (cards, hover states)
  • --text-primary: Main text colour
  • --text-secondary: Secondary text (labels, notes)
  • --text-tertiary: Tertiary text (verse numbers, disabled)
  • --accent-primary: Primary action colour (buttons, links)
  • --accent-positive: Success/positive actions
  • --accent-negative: Errors/warnings
  • --accent-info: Info/notes
  • --highlight-orange: Fifth highlight colour
  • --highlight-pink: Sixth highlight colour
